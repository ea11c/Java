When exectued with no command line arguments FileViewer displays a list of all files and directories in your current directory along with their size (* for directories)

-i [<file>|<directory>]
	displays information about the specified file, or lists all files and directories in the specified directory.  Defaults to current directory

-v <file>
	displays contents of a file on the screen

-c <sourceFile> <destFile>
	copies the contents from the sourceFile to the destFile
